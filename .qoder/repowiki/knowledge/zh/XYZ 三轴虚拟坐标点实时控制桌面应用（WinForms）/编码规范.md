- 自定义控件统一继承 Control 并重写 OnPaint，构造函数中设置 DoubleBuffered=true、ResizeRedraw=true，OnPaint 开头调用 PaintHelper.SetupGraphics(g)。
- 所有尺寸基于 ClientRectangle 相对计算，禁止硬编码绝对像素值；状态变化后调用 Invalidate() 触发重绘而非阻塞 UI 线程。
- 为兼容 C# 5，事件订阅不使用 Lambda，统一采用命名方法 + 显式委托构造（new EventHandler(...)）。
- 双向数据绑定（如滑块↔数字框）通过 _syncing 标志位防止回调递归触发。
- UI 与业务解耦：MainForm 只做输入捕获→转发 Hub→监听 Changed→刷新控件，业务算法全部下沉至 Logic/ 层以便测试。
