解决方案由四个工程组成，依赖方向严格单向：
- `src/XyzController`（主应用）：Program.cs 入口 → MainForm.cs Facade 协调器 → Logic/ 层（AxisController、AxisJogService、XyzControllerHub）。
- `src/XyzController.Controls`（控件库）：纯 GDI+ 手绘自定义控件（JoystickPad、XYView、ZBarView、AxisBar、DroLabel、JogButton），仅依赖 MathHelper/PaintHelper 工具类。
- `src/XyzController.Tests`（测试工程）：自托管轻量测试框架（TestAttribute / Assert / TestRunner）+ 单元测试，独立于 UI，只依赖逻辑层。
- `src/XyzController.WpfHost`（WPF 宿主）：可选 WPF 容器以承载 WinForms 控件。
核心通信通过 XyzControllerHub 事件总线协调 AxisController、AxisJogService 与 UI 之间的状态同步；MainForm 作为 Facade 仅做输入转发与刷新，不承载业务算法。
