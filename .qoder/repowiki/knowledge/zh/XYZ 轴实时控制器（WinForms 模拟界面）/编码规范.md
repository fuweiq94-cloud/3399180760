- 业务类（AxisController、XyzControllerHub、AxisJogService）均不引用 System.Windows.Forms/System.Drawing，保持 UI 无关以便单独测试与跨场景复用。
- 状态变更通过 protected virtual OnChanged() 触发 EventHandler 事件，子类覆写扩展或基类转发（如 Hub 转发三轴的 Changed）。
- 数值边界一律通过 Controls/MathHelper.Clamp* 静态方法钳制，避免在各处重复 if 判断。
- 动画采用“目标值 vs 当前值”双缓冲模型：UI 只修改 Target，由 Advance(lerpFraction) 每帧插值逼近 Current，并在剩余距离小于阈值时吸附到位。
- 为兼容旧版 csc 编译器，委托订阅使用命名方法 + 显式委托构造（`new EventHandler(...)`）而非 Lambda 表达式。
