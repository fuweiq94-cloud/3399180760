解决方案由四个工程组成：`XyzController`（主应用，含 Program.cs / MainForm.cs 与 Logic/ 业务层）、`XyzController.Controls`（可复用 WinForms 控件库，纯 GDI+ 手绘）、`XyzController.Tests`（自托管轻量测试框架 + 单元测试）以及 `XyzController.WpfHost`（WPF 宿主）。依赖方向严格单向：主应用 → 控件库 + 逻辑层；控件库仅依赖 MathHelper/PaintHelper 工具类；测试工程独立于 UI，只依赖逻辑层。核心通信通过 `Logic/XyzControllerHub.cs` 事件总线协调 AxisController、AxisJogService 与 UI 之间的状态同步，MainForm 作为 Facade 仅做输入转发与刷新，不承载业务算法。
