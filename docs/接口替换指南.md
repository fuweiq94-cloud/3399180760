# 轴控制接口替换指南

## 1. 概述

当前 XyzController 项目使用纯软件模拟的轴控制（AxisController + XyzControllerHub），通过 animTimer 驱动插值动画。本指南说明如何将软件模拟后端替换为实际机械手 API 调用，同时保持 UI 层和 Logic 层接口不变。

### 1.1 现有架构

```
UI (MainForm)  →  XyzControllerHub  →  AxisController（软件插值模拟）
                        ↑
                 AxisJogService
```

### 1.2 目标架构

```
UI (MainForm)  →  XyzControllerHub  →  AxisController（接口不变）
                        ↑                      ↓
                 AxisJogService        IRobotAdapter（适配层）
                                              ↓
                                    实际机械手 API（如 xyzu机械手）
```

### 1.3 设计原则
- **UI 层零改动**：MainForm 和自定义控件不需要任何修改
- **Hub 接口不变**：XyzControllerHub 的 SetTarget/Advance/Changed 等方法签名保持不变
- **适配层隔离**：新增 IRobotAdapter 接口和具体实现，隔离机械手 API 细节
- **可切换**：软件模拟和实际机械手可通过配置切换

## 2. 机械手 API 调用格式

根据实际机械手的 API，典型的调用方式如下：

### 2.1 绝对定位

```
xyzu机械手.abs(轴标识, 目标位置, 速度)
```

示例：
```
xyzu机械手.abs(轴X, 50.0, 慢速)     // X轴绝对定位到50.0，慢速
xyzu机械手.abs(轴Y, -30.0, 慢速)    // Y轴绝对定位到-30.0，慢速
xyzu机械手.abs(轴Z, 10.0, 慢速)     // Z轴绝对定位到10.0，慢速
```

### 2.2 相对定位（步进）

```
xyzu机械手.rel(轴标识, 增量, 速度)
```

示例：
```
xyzu机械手.rel(轴X, 5.0, 中速)     // X轴正向步进5.0
xyzu机械手.rel(轴X, -5.0, 中速)    // X轴负向步进5.0
```

### 2.3 回原点

```
xyzu机械手.home(轴标识, 速度)
```

示例：
```
xyzu机械手.home(轴X, 慢速)
xyzu机械手.home(轴Y, 慢速)
xyzu机械手.home(轴Z, 慢速)
```

### 2.4 急停

```
xyzu机械手.estop()
```

### 2.5 读取当前位置

```
float pos = xyzu机械手.pos(轴标识)
```

### 2.6 速度档位映射

| 机械手速度 | Hub.SpeedSetting | 说明 |
|-----------|-----------------|------|
| 慢速 | 0-20 | 精确定位、手动操作 |
| 中速 | 21-60 | 正常运行 |
| 快速 | 61-100 | 大范围移动 |

## 3. 适配层设计

### 3.1 IRobotAdapter 接口

```csharp
using System;

namespace XyzController.Logic
{
    /// <summary>
    /// 机械手适配器接口。
    /// 软件模拟和实际机械手均实现此接口，实现可切换后端。
    /// </summary>
    public interface IRobotAdapter
    {
        /// <summary>
        /// 绝对定位：将指定轴移动到目标位置
        /// </summary>
        /// <param name="axisName">轴名称（"X"/"Y"/"Z"）</param>
        /// <param name="position">目标位置</param>
        /// <param name="speed">速度档位 [0-100]</param>
        void MoveAbs(string axisName, float position, int speed);

        /// <summary>
        /// 相对定位：在当前基础上步进
        /// </summary>
        /// <param name="axisName">轴名称</param>
        /// <param name="delta">增量（正=正向，负=负向）</param>
        /// <param name="speed">速度档位 [0-100]</param>
        void MoveRel(string axisName, float delta, int speed);

        /// <summary>
        /// 回原点
        /// </summary>
        /// <param name="axisName">轴名称</param>
        /// <param name="speed">速度档位</param>
        void Home(string axisName, int speed);

        /// <summary>
        /// 急停所有轴
        /// </summary>
        void EmergencyStop();

        /// <summary>
        /// 读取轴的当前位置
        /// </summary>
        /// <param name="axisName">轴名称</param>
        /// <returns>当前位置</returns>
        float GetPosition(string axisName);

        /// <summary>
        /// 速度档位转换为机械手速度标识
        /// </summary>
        /// <param name="speedSetting">速度档位 [0-100]</param>
        /// <returns>速度标识字符串</returns>
        string SpeedToString(int speedSetting);
    }
}
```

### 3.2 软件模拟适配器（当前行为的封装）

```csharp
using System;

namespace XyzController.Logic
{
    /// <summary>
    /// 软件模拟适配器。
    /// 保持原有的 animTimer 插值行为，不做实际硬件通信。
    /// </summary>
    public class SimulationAdapter : IRobotAdapter
    {
        private readonly XyzControllerHub _hub;

        public SimulationAdapter(XyzControllerHub hub)
        {
            _hub = hub;
        }

        public void MoveAbs(string axisName, float position, int speed)
        {
            // 软件模拟：直接设置 Target，由 animTimer 插值逼近
            switch (axisName)
            {
                case "X": _hub.X.SetTarget(position); break;
                case "Y": _hub.Y.SetTarget(position); break;
                case "Z": _hub.Z.SetTarget(position); break;
            }
        }

        public void MoveRel(string axisName, float delta, int speed)
        {
            switch (axisName)
            {
                case "X": _hub.X.Step((int)delta); break;
                case "Y": _hub.Y.Step((int)delta); break;
                case "Z": _hub.Z.Step((int)delta); break;
            }
        }

        public void Home(string axisName, int speed)
        {
            switch (axisName)
            {
                case "X": _hub.X.ResetToOrigin(); break;
                case "Y": _hub.Y.ResetToOrigin(); break;
                case "Z": _hub.Z.ResetToOrigin(); break;
            }
        }

        public void EmergencyStop()
        {
            // 软件模拟：Target = Current
            _hub.X.SetTarget(_hub.X.Current);
            _hub.Y.SetTarget(_hub.Y.Current);
            _hub.Z.SetTarget(_hub.Z.Current);
        }

        public float GetPosition(string axisName)
        {
            switch (axisName)
            {
                case "X": return _hub.X.Current;
                case "Y": return _hub.Y.Current;
                case "Z": return _hub.Z.Current;
                default: return 0f;
            }
        }

        public string SpeedToString(int speedSetting)
        {
            if (speedSetting <= 20) return "慢速";
            if (speedSetting <= 60) return "中速";
            return "快速";
        }
    }
}
```

## 4. Hub 改造（可选）

如果希望 Hub 内部使用适配器而非直接操作 AxisController，可以新增一个带适配器的 Hub 扩展：

```csharp
using System;

namespace XyzController.Logic
{
    /// <summary>
    /// 带机械手适配器的 Hub 扩展。
    /// 在保持 XyzControllerHub 接口不变的前提下，
    /// 将 SetTarget 等操作转发到 IRobotAdapter。
    /// </summary>
    public class RobotHub : XyzControllerHub
    {
        private readonly IRobotAdapter _adapter;

        public RobotHub(IRobotAdapter adapter,
                        float xMin, float xMax,
                        float yMin, float yMax,
                        float zMin, float zMax)
            : base(xMin, xMax, yMin, yMax, zMin, zMax)
        {
            _adapter = adapter;
        }

        /// <summary>
        /// 重写 SetTarget：同时设置软件 Target 和调用机械手 API
        /// </summary>
        public new void SetTarget(float x, float y, float z)
        {
            // 1. 保持软件模拟同步（用于 UI 显示）
            base.SetTarget(x, y, z);

            // 2. 调用机械手 API
            int speed = SpeedSetting;
            _adapter.MoveAbs("X", x, speed);
            _adapter.MoveAbs("Y", y, speed);
            _adapter.MoveAbs("Z", z, speed);
        }
    }
}
```

## 5. 替换步骤

### 步骤 1：创建适配器接口和实现
1. 在 `Logic/` 目录下新建 `IRobotAdapter.cs`
2. 创建 `SimulationAdapter.cs`（封装当前软件模拟行为）
3. 创建 `XyzuRobotAdapter.cs`（封装实际机械手 API）

### 步骤 2：在 MainForm 中切换适配器
```csharp
// 原来（纯软件模拟）：
_hub = new XyzControllerHub(-100f, 100f, -100f, 100f, 0f, 50f);

// 替换为（使用实际机械手）：
var adapter = new XyzuRobotAdapter();
_hub = new RobotHub(adapter, -100f, 100f, -100f, 100f, 0f, 50f);
```

### 步骤 3：修改 AxisJogService（可选）
如果需要 JOG 操作也通过机械手 API，可以扩展 AxisJogService：

```csharp
// 在 AxisJogService 中注入适配器
public class AxisJogService
{
    private readonly IRobotAdapter _adapter;
    
    // OnJogStart 中调用：
    // 寸动模式：_adapter.MoveRel(axisName, direction * stepDistance, speed)
    // 连续模式：_adapter.MoveAbs(axisName, direction > 0 ? Max : Min, speed)
}
```

### 步骤 4：位置反馈（替换 animTimer）
使用机械手时，Current 应来自编码器反馈而非软件插值：

```csharp
// 原来由 animTimer 驱动：
_hub.Advance();  // 软件插值

// 替换为从机械手读取实际位置：
_hub.X.SetCurrentFromHardware(_adapter.GetPosition("X"));
_hub.Y.SetCurrentFromHardware(_adapter.GetPosition("Y"));
_hub.Z.SetCurrentFromHardware(_adapter.GetPosition("Z"));
```

需要在 AxisController 中新增方法：
```csharp
/// <summary>
/// 从硬件读取的实际位置更新 Current（仅用于硬件模式）
/// </summary>
public void SetCurrentFromHardware(float value)
{
    _current = MathHelper.Clamp(value, Min, Max);
    OnChanged();
}
```

## 6. 调用对照表

| 当前接口调用 | 机械手 API 等价调用 | 说明 |
|-------------|-------------------|------|
| `hub.SetTarget(50, -30, 10)` | `xyzu机械手.abs(轴X, 50, 慢速)`<br>`xyzu机械手.abs(轴Y, -30, 慢速)`<br>`xyzu机械手.abs(轴Z, 10, 慢速)` | 三轴绝对定位 |
| `hub.X.Step(5)` | `xyzu机械手.rel(轴X, 5, 中速)` | X轴正向步进 |
| `hub.X.Step(-5)` | `xyzu机械手.rel(轴X, -5, 中速)` | X轴负向步进 |
| `hub.ResetToOrigin()` | `xyzu机械手.home(轴X, 慢速)`<br>`xyzu机械手.home(轴Y, 慢速)`<br>`xyzu机械手.home(轴Z, 慢速)` | 三轴回原点 |
| `jogService.OnJogStart(+1)` (寸动) | `xyzu机械手.rel(轴X, 步长, 慢速)` | 寸动正向 |
| `jogService.OnJogStart(-1)` (连续) | `xyzu机械手.abs(轴X, Max, 快速)` | 连续正向 |
| `jogService.OnJogStop()` | `xyzu机械手.abs(轴X, 当前位置, 慢速)` | 停止（冻结到当前位置） |
| `jogService.EmergencyStop()` | `xyzu机械手.estop()` | 急停 |
| `axis.Current` | `xyzu机械手.pos(轴X)` | 读取当前位置 |
| `hub.SpeedSetting = 20` | 速度参数 = `慢速` | 速度设置 |

## 7. 注意事项

1. **线程安全**：机械手 API 调用可能在专用通信线程中执行，与 UI 线程不同。需要使用 `Invoke` 或 `Dispatcher.Invoke` 回到 UI 线程更新界面。
2. **错误处理**：机械手通信可能失败（超时、断线），需要添加异常捕获和重连机制。
3. **实时性**：.NET Framework + WinForms 非实时平台，如果机械手要求严格周期控制，建议使用独立线程。
4. **限位保护**：机械手 API 调用前应再次检查目标值在物理限位范围内。
5. **回退能力**：保留 SimulationAdapter，在无机械手时仍可软件模拟运行。

## 8. 点动（JOG）控制代码改造

点动（JOG）是工业控制中最常用的操作方式之一。本节详细说明如何将现有的软件模拟 JOG 改造为通过机械手 API 执行的实际 JOG 操作。

### 8.1 现有 JOG 架构分析

当前 JOG 控制涉及三层代码：

```
JogButton 控件（UI 层）
    ↓ Jog/Stop 事件
MainForm（协调层）
    ↓ JogButton_Jog / JogButton_Stop
AxisJogService（Logic 层）
    ↓ OnJogStart / OnJogStop
AxisController（数据层）
    → SetTarget / Step
```

**现有 AxisJogService 的核心逻辑**：

```csharp
// 寸动模式 OnJogStart：
axis.SetTarget(axis.Target + direction * StepDistance);

// 连续模式 OnJogStart：
axis.SetTarget(direction > 0 ? axis.Max : axis.Min);

// 连续模式 OnJogStop：
axis.SetTarget(axis.Current);  // 冻结到当前位置
```

### 8.2 改造方案：为 AxisJogService 注入适配器

#### 8.2.1 修改 AxisJogService 构造函数

```csharp
using System;

namespace XyzController.Logic
{
    public class AxisJogService
    {
        private readonly AxisController _axis;
        private readonly IRobotAdapter _adapter;  // 新增：机械手适配器
        private JogMode _mode = JogMode.Incremental;
        private float _stepDistance = 1.0f;
        private bool _isJogging;
        private int _speed = 20;  // 新增：JOG 速度档位

        // 原有构造函数（保持向后兼容，使用软件模拟适配器）
        public AxisJogService(AxisController axis)
            : this(axis, null)
        {
        }

        // 新增：带适配器的构造函数
        public AxisJogService(AxisController axis, IRobotAdapter adapter)
        {
            if (axis == null)
                throw new ArgumentNullException("axis");
            _axis = axis;
            _adapter = adapter;  // null 表示纯软件模拟
        }

        // 新增：设置 JOG 速度
        public void SetSpeed(int speed)
        {
            _speed = speed;
        }
    }
}
```

#### 8.2.2 改造 OnJogStart 方法

```csharp
public void OnJogStart(int direction)
{
    if (direction != 1 && direction != -1)
        throw new ArgumentException("direction must be +1 or -1");

    _isJogging = true;

    if (_mode == JogMode.Incremental)
    {
        // === 寸动模式 ===
        if (_adapter != null)
        {
            // 机械手模式：调用 rel() 相对定位
            // 对应：xyzu机械手.rel(轴X, 方向*步长, 慢速)
            _adapter.MoveRel(_axis.Name, direction * _stepDistance, _speed);
        }
        
        // 软件模拟同步（用于 UI 显示）
        _axis.SetTarget(_axis.Target + direction * _stepDistance);
    }
    else
    {
        // === 连续模式 ===
        if (_adapter != null)
        {
            // 机械手模式：调用 abs() 移动到限位
            // 对应：xyzu机械手.abs(轴X, Max, 快速)  或  xyzu机械手.abs(轴X, Min, 快速)
            float limit = direction > 0 ? _axis.Max : _axis.Min;
            _adapter.MoveAbs(_axis.Name, limit, _speed);
        }
        
        // 软件模拟同步
        _axis.SetTarget(direction > 0 ? _axis.Max : _axis.Min);
    }
}
```

#### 8.2.3 改造 OnJogStop 方法

```csharp
public void OnJogStop()
{
    if (!_isJogging)
        return;

    _isJogging = false;

    if (_mode == JogMode.Continuous)
    {
        if (_adapter != null)
        {
            // 机械手模式：冻结到当前位置
            // 对应：xyzu机械手.abs(轴X, 当前位置, 慢速)
            float currentPos = _adapter.GetPosition(_axis.Name);
            _adapter.MoveAbs(_axis.Name, currentPos, _speed);
        }
        
        // 软件模拟同步
        _axis.SetTarget(_axis.Current);
    }
    // 寸动模式 OnJogStop 无操作（走完一步自动停）
}
```

#### 8.2.4 改造 EmergencyStop 方法

```csharp
public void EmergencyStop()
{
    _isJogging = false;

    if (_adapter != null)
    {
        // 机械手急停
        // 对应：xyzu机械手.estop()
        _adapter.EmergencyStop();
    }
    
    // 软件模拟同步
    _axis.SetTarget(_axis.Current);
}
```

### 8.3 JogButton 控件层改造

JogButton 控件本身不需要修改——它只负责发出 `Jog` 和 `Stop` 事件。改造点在 MainForm 的事件处理中。

#### 8.3.1 MainForm 中创建带适配器的 JOG 服务

```csharp
// 原来（纯软件模拟）：
_jogX = new AxisJogService(_hub.X);
_jogY = new AxisJogService(_hub.Y);
_jogZ = new AxisJogService(_hub.Z);

// 改造后（使用机械手适配器）：
IRobotAdapter adapter = new XyzuRobotAdapter();
_jogX = new AxisJogService(_hub.X, adapter);
_jogY = new AxisJogService(_hub.Y, adapter);
_jogZ = new AxisJogService(_hub.Z, adapter);

// 设置 JOG 速度
_jogX.SetSpeed(20);  // 慢速
_jogY.SetSpeed(20);
_jogZ.SetSpeed(20);
```

#### 8.3.2 JogButton 事件处理（无需改动）

```csharp
// 这部分代码不需要修改，因为 JogButton 只发事件，实际执行在 AxisJogService 中
private void JogButton_Jog(object sender, JogEventArgs e)
{
    // e.Direction: +1 或 -1
    // e.AxisName: "X" / "Y" / "Z"
    switch (e.AxisName)
    {
        case "X": _jogX.OnJogStart(e.Direction); break;
        case "Y": _jogY.OnJogStart(e.Direction); break;
        case "Z": _jogZ.OnJogStart(e.Direction); break;
    }
}

private void JogButton_Stop(object sender, JogEventArgs e)
{
    switch (e.AxisName)
    {
        case "X": _jogX.OnJogStop(); break;
        case "Y": _jogY.OnJogStop(); break;
        case "Z": _jogZ.OnJogStop(); break;
    }
}
```

```csharp
private IRobotAdapter _adapter;  // 机械手适配器
private int _keyboardStep = 1;    // 键盘步进步长

private void MainForm_KeyDown(object sender, KeyEventArgs e)
{
    bool big = (e.Modifiers & Keys.Shift) == Keys.Shift;
    int step = big ? 10 : _keyboardStep;
    int speed = big ? 60 : 20;  // Shift = 中速，普通 = 慢速

    switch (e.KeyCode)
    {
        case Keys.W:
            // 原来：_hub.Y.Step(big ? 10 : 1)
            // 改造后：通过适配器执行相对定位
            if (_adapter != null)
                _adapter.MoveRel("Y", step, speed);
            _hub.Y.Step(step);  // 软件模拟同步
            break;
        case Keys.S:
            if (_adapter != null)
                _adapter.MoveRel("Y", -step, speed);
            _hub.Y.Step(-step);
            break;
        case Keys.A:
            if (_adapter != null)
                _adapter.MoveRel("X", -step, speed);
            _hub.X.Step(-step);
            break;
        case Keys.D:
            if (_adapter != null)
                _adapter.MoveRel("X", step, speed);
            _hub.X.Step(step);
            break;
        case Keys.Space:
            if (_adapter != null)
                _adapter.MoveRel("Z", step, speed);
            _hub.Z.Step(step);
            break;
        case Keys.Escape:
            // 急停
            if (_adapter != null)
                _adapter.EmergencyStop();
            _jogX.EmergencyStop();
            _jogY.EmergencyStop();
            _jogZ.EmergencyStop();
            break;
    }
}
```

### 8.5 JOG 模式与机械手 API 的完整对照表

| 操作场景 | 现有代码 | 机械手 API 调用 | 说明 |
|---------|---------|----------------|------|
| 寸动按下（正向） | `axis.SetTarget(Target + step)` | `xyzu机械手.rel(轴X, +步长, 慢速)` | 走一个步长 |
| 寸动按下（负向） | `axis.SetTarget(Target - step)` | `xyzu机械手.rel(轴X, -步长, 慢速)` | 反向走一个步长 |
| 寸动松开 | 无操作 | 无操作 | 走完自动停 |
| 连续按下（正向） | `axis.SetTarget(Max)` | `xyzu机械手.abs(轴X, Max, 快速)` | 持续走到限位 |
| 连续按下（负向） | `axis.SetTarget(Min)` | `xyzu机械手.abs(轴X, Min, 快速)` | 反向持续走 |
| 连续松开 | `axis.SetTarget(Current)` | `xyzu机械手.abs(轴X, 当前位置, 慢速)` | 冻结到当前位置 |
| 急停 | `axis.SetTarget(Current)` | `xyzu机械手.estop()` | 立即停止所有轴 |
| 键盘 W（前） | `hub.Y.Step(1)` | `xyzu机械手.rel(轴Y, +1, 慢速)` | Y轴正向步进 |
| 键盘 S（后） | `hub.Y.Step(-1)` | `xyzu机械手.rel(轴Y, -1, 慢速)` | Y轴负向步进 |
| 键盘 A（左） | `hub.X.Step(-1)` | `xyzu机械手.rel(轴X, -1, 慢速)` | X轴负向步进 |
| 键盘 D（右） | `hub.X.Step(1)` | `xyzu机械手.rel(轴X, +1, 慢速)` | X轴正向步进 |
| 键盘 Space（上） | `hub.Z.Step(1)` | `xyzu机械手.rel(轴Z, +1, 慢速)` | Z轴正向步进 |
| Shift+W（大步） | `hub.Y.Step(10)` | `xyzu机械手.rel(轴Y, +10, 中速)` | Y轴大步正向 |
| Esc（急停） | 三轴冻结 | `xyzu机械手.estop()` | 全轴急停 |
